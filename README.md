# Loan Calculator App

A clean, functional Python GUI that calculates monthly and total loan payments based on user input. Built with Tkinter, itâ€™s designed to be simple, educational, and easy to expand.

## Features (Version 1.0)
- User input for loan amount, interest rate, and term
- Calculates monthly payment, total payment, and total interest
- Input validation and error handling
- Clean interface styling and readable results
- Real-time calculation with reset functionality

## Technologies Used
- Python 3.x
- Tkinter
